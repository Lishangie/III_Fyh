# Quick Start - HW05

## Подготовка

```bash
cd homeworks/HW05
pip install -r requirements.txt
python prepare_data.py
```

## Открыть ноутбук

```bash
jupyter notebook notebook.ipynb
```

## Запустить анализ

```bash
python main.py
```

## Файлы

- `notebook.ipynb` - майн анализ
- `advanced_analysis.ipynb` - расширенный анализ
- `main.py` - скрипт выполнения
- `models.py` - МЛ-модели
- `utils.py` - вспомогательные функции
- `config.json` - настройки
